
# Crypto Data Pipeline - Домашна задача 1

## Опис на проектот
Ова е имплементација на Pipe and Filter архитектура за обработка на крипто податоци од берзи. Системот автоматски презема, трансформира и складира историски податоци за 1000 највредни криптовалути.

## Структура на проектот 


## Како да се користи
1. Инсталирај ги зависностите: `pip install -r requirements.txt`
2. Пушти ја главната програма: `python main.py`

## Податоци
- **883 криптовалути** со пазарна капитализација
- **665 историски датотеки** со OHLC податоци  
- **4-10 години** историски податоци по криптовалута
- **CSV формат** за сите податоци

## Технологии
- Python, Pandas, Requests, yfinance
- CoinGecko API, Yahoo Finance API
- Pipe and Filter архитектура
